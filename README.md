# ✨ Elementum Agency Website

A modern, responsive agency landing page built using **React (Vite + TypeScript)** as part of an internship assignment.

This project is a **pixel-perfect implementation of a Figma design**, developed using a structured UI development approach (vibe coding style).

## 📌 Project Overview

Elementum Agency Website is a fully responsive landing page designed for a digital agency.  
It focuses on modern UI/UX principles, clean layout structure, and reusable React components.

The goal of this project was to convert a Figma design into a fully functional web application with high accuracy in:

- Spacing
- Typography
- Layout alignment
- Responsive behavior

This project was built using a **vibe coding workflow**, where the UI was developed iteratively:

- Breaking Figma into components
- Building section by section
- Adjusting spacing and typography for accuracy
- Refining responsiveness for all screen sizes
- Ensuring UI matches design expectations

## 📦 Installation Guide

### Clone the repository
```bash
git clone https://github.com/your-username/elementum-agency.git
```

### Navigate to project folder
```bash
cd elementum-agency
```

### Install dependencies
```bash
npm install
```

or

```bash
yarn install
```

---

## ▶️ Run the Project

Start development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:5173
```

---

## 🏗️ Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
